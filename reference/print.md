# Print CAST classes

Generic print function for trainDI and aoa

## Usage

``` r
# S3 method for class 'trainDI'
print(x, ...)

show.trainDI(x, ...)

# S3 method for class 'aoa'
print(x, ...)

show.aoa(x, ...)

# S3 method for class 'nndm'
print(x, ...)

show.nndm(x, ...)

# S3 method for class 'knndm'
print(x, ...)

show.knndm(x, ...)

# S3 method for class 'ffs'
print(x, ...)

show.ffs(x, ...)
```

## Arguments

- x:

  An object of type *ffs*

- ...:

  other arguments.
